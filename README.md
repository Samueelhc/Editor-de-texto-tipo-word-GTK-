# 📝 Editor de Texto en C con GTK

Este proyecto consiste en un **editor de texto básico desarrollado en C utilizando la biblioteca GTK** para la interfaz gráfica. Fue realizado como **proyecto académico** y permite escribir, editar y guardar texto, además de incluir funciones adicionales como **encriptado y desencriptado de contenido usando logica de matrices y corrección ortográfica**.

El programa implementa una interfaz gráfica simple donde el usuario puede escribir texto, manipularlo mediante diferentes botones y guardar el contenido en archivos `.txt`.

---

## 🚀 Funcionalidades

- Interfaz gráfica desarrollada con **GTK**.
- Área de edición de texto para escribir y modificar contenido maximo 4 paginas.
- Guardado del texto en **archivos `.txt`**.
- **Corrector ortográfico integrado** mediante diccionarios.
- Botón para **borrar el contenido del editor**.
- Ventana de **ayuda para el usuario**.
- Ventana de confirmación para **salir de la aplicación**.
- Sistema de **encriptado y desencriptado de texto**.
- Manejo de eventos y botones dentro de la interfaz.
- Control de desplazamiento automático dentro del editor.

---

## 🛠️ Tecnologías Utilizadas

- **C**
- **GTK** (interfaz gráfica)
- **Hunspell** (corrector ortográfico)
- Manejo de archivos `.txt`

---

## 🎯 Objetivo del Proyecto

Desarrollar un **editor de texto con interfaz gráfica** aplicando conceptos de programación en C, manejo de bibliotecas externas, eventos de interfaz gráfica, corrección ortográfica y manipulación de texto dentro de una aplicación de escritorio.
